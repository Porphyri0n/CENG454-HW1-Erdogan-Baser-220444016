## Project Overview
This repository contains the first homework assignment focusing on the "Agile Flight Pipeline". It demonstrates a functional flying Aero-Prototype built in Unity 6, utilizing proper Git workflows, Agile methodology (Kanban), and Git LFS for binary asset management.

## Features
* **3-Axis Flight Control:** Pitch, yaw, and roll mechanics implemented using `Time.deltaTime` for frame-rate independence.
* **Git LFS Integration:** High-resolution 3D aircraft models and binary assets are properly tracked.
* **Agile Workflow:** Developed using a strict feature-branch workflow and Pull Request reviews.

## Requirements
* Unity 6
* Git LFS
