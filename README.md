# CENG 454 HW1 - Sky-High Prototype

**Author:** Erdoğan Başer (220444016)  
**Course:** CENG 454: Game Programming - Spring 2025-2026

## Project Overview
This repossitory contains CENG454 Homework 1 project files. Homework's topic is creating a basic protoype of a 3D flight controller.

## Features
* **3-Axis Flight Control:** Pitch, yaw, and roll mechanics created using `Time.deltaTime` for frame-rate independence.
* **Git LFS Integration:** High-resolution 3D aircraft models and binary assets are added to project.
* **Agile Workflow:** Developed using a feature-branch workflow and Pull Request reviews.

## Requirements
* Unity 6.3
* Git LFS
